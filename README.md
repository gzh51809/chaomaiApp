# 超买网APP
## 作者：卢俊洪

## 项目架构

>开发IDE：VScode

>前端开发：React框架 + Redux框架 + antd-mobile框架 + webpack + ECMAScript6 + scss

>后端开发：

>（NodeJS）使用11.1.0（ECMAScript6） + mongodb4.0.4

## 项目演示地址
